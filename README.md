# Pro Front-End Portfolio (Next.js + Tailwind)

## Run locally
1) Install deps:
```bash
npm i
```
2) Start dev server:
```bash
npm run dev
```

## Edit your content fast
- **Your name + links:** `components/navbar.tsx` and `components/hero.tsx`
- **Projects list:** `data/projects.ts`
- **Experience bullets:** `components/experience.tsx`
- **Contact email:** `components/contact.tsx`

## Add your resume
Edit your name, bio, links, and contact info in `data/profile.ts`.

## Deploy
- Push to GitHub
- Import in Vercel
- Set domain + update `metadataBase` in `app/layout.tsx`
